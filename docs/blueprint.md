# **App Name**: GoalFlow

## Core Features:

- Create Goal: Create a new goal with a mandatory name.
- List Goals: List all existing goals.
- Edit Goal: Edit the name of an existing goal.
- Delete Goal: Delete a goal with confirmation, also deletes all associated tasks.
- Add Task to Goal: Add a task with a mandatory title to a specific goal.
- List Tasks for Goal: List all tasks associated with a specific goal.
- Calculate Progress: Calculate the percentage of completed tasks for each goal.

## Style Guidelines:

- Primary color: Deep purple (#6750A4) to evoke focus and determination.
- Background color: Light lavender (#E9DFF7), a desaturated version of the primary color to provide a calm backdrop.
- Accent color: Dusty rose (#A44A5B), offering a contrasting visual element for completed tasks and key actions.
- Body and headline font: 'Inter', a sans-serif font, for a modern and objective feel.
- Use minimalist icons to represent goals and tasks, focusing on clarity and simplicity.
- Employ a clean, card-based layout to organize goals and tasks efficiently.
- Implement subtle animations, like progress bar updates, to provide dynamic visual feedback.
